Elenco dei file
(INCSTRAD) Rilevazione degli incidenti stradali con lesioni a persone
Anno 2018

Il file ISTAT_MFR_INCSTRAD_2018_IT.zip contiene, in due directory dedicate, i file di seguito elencati:
file di microdati di esempio, tracciato record con classificazioni associate, e documenti metodologici.
Il nome di ciascun file è caratterizzato dalla tipologia del file (MFR), dall'acronimo dell'indagine, dal contenuto dello stesso, dal periodo di riferimento dei dati e dall'eventuale area tematica.
Il file di microdati di esempio contiene dati di prova ed ha il formato con campi separati dal carattere '^'
I dati sono generati casualmente e non sono utilizzabili per elaborazioni. I dati di esempio contengono tutte le variabili del file di microdati originali con gli stessi nomi e valori di classificazione.
Per quanto riguarda le classificazioni, esse sono raggruppate nella cartella Classificazioni. Il nome del file di ciascuna classificazione è contrassegnato dal suffisso var seguito dal numero d'ordine presente nel tracciato record relativo alle variabili di tipo categorico.
